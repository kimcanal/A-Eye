# Module 1 Asset Hook

This folder is the placeholder for the provided 3D assets used by the assignment.

## Expected asset types
- Map asset: city road network / intersections
- Object assets: taxis, regular vehicles, autonomous vehicle, obstacles, passengers
- Animation assets: movement / traffic flow animations

## Current status
- The actual 3D assets are not bundled in this repository.
- This folder exists so the Module 1 scaffold can point to where the assets will be placed.
- The runnable minimal simulation uses a simplified 2D/state-log representation until the 3D assets are integrated.
