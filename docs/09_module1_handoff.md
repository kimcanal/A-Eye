# Module 1 Handoff

## What to do next
1. Open Unity and create a project.
2. Import the provided Kakaomobility `.unitypackage` files.
3. Build the 3x3 city scene using `street_main`.
4. Place `car_taxi` and obstacle objects.
5. Add `carwheel_move` and `kickboard_fall` animations.
6. Connect the scene events to the Module 1 scenario model.

## What the code does now
- `minimal_simulation.py` runs a tiny city and writes JSON logs.
- `minimal_stub.py` stores the scenario blueprint.

## What the asset pack gives us
- Map asset
- Taxi object
- Obstacle objects
- Animation assets
- Map-setting asset

## Plain-language summary
Module 1 is the stage where we build the city and make it move a little. Once the real Unity assets are imported, this stage can become the actual digital twin scene.
