# Module 1 Code Map

## Simple view
Module 1 is a tiny fake city that can move a little and write down what happened.

## Files
- `module1_simulation/minimal_simulation.py` → runnable simulation
- `module1_simulation/minimal_stub.py` → scenario blueprint
- `module1_simulation/README.md` → how to run it
- `module1_simulation/scenario.md` → city story and object counts
- `module1_simulation/asset_requirements.md` → 3D asset shopping list
- `module1_simulation/integration_plan.md` → how to connect to real Unity assets
- `module1_simulation/assets/README.md` → asset drop folder
- `docs/01_module1_explainer.md` → beginner-friendly explanation
- `docs/02_asset_pack_analysis.md` → downloaded asset pack analysis
- `docs/03_unity_import_checklist.md` → Unity import steps
- `docs/04_module1_results.md` → output result interpretation
- `docs/05_module1_final_summary.md` → short final summary
- `docs/06_team_guide_module1.md` → who does what next
- `docs/07_unity_workflow_checklist.md` → Unity task checklist
- `docs/01_module1_compendium.md` → all Module 1 work in one place
